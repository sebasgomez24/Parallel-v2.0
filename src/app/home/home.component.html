Augmented Reality (AR) is the use of computer generated images or data to modify the real world. AR uses the existing environment and overlays new information on top of it.

Virtual Reality (VR) is the use of computer technologies to create a simulated world. Users are immersed within the built environment and able to move about the 3D models.
